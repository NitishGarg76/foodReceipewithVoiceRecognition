
# Getting Started


## Setup For Android

1. Clone the repository:
   git clone <repository_url>
   cd ChatbotApp
2. npm install
3. Start Metro - npm start
4. run command - npm run android

## Setup For IOS

1. cd IOS
2. run command - pod insall --repo-update
3. run command - npm run ios
